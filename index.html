Base de Datos Personal (Firebase)

Si abres esta página en tu computadora (fuera de esta plataforma), puedes conectar tu propio proyecto gratuito de Firebase Firestore para seguir teniendo reservas compartidas en línea y tiempo real de manera independiente.

¿Cómo obtener estos datos?
  1. Entra a Firebase Console y crea un proyecto gratuito.
  2. Crea una base de datos "Cloud Firestore" en modo de prueba.
  3. Añade una aplicación Web y copia el objeto firebaseConfig que te proporcionen.
  4. Pégalo aquí y presiona "Guardar y Conectar".
